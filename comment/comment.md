## Comments [**Back**](./../README.md)

- use `/** ... */` for a single line
- use follow for multi-line:

```js
/**
 * this is a mutli-line comment
 */
```

#### 1. Function comments

```js
/**
 * [initData: description]
 * @param  {[type]} data [description]
 * @return {[type]}      [description]
 */
function initData(data) {
    /** ... */
}
```
