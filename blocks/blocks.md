#### 1. Use braces

- To use braces with all multiple lines, but do not use when it's a single line.

```js
/**
 * bad
 */
if (test)
    return false;

/**
 * good
 */
if (test) return false;
if (test) {
    return false;
}

/**
 * bad
 */

```
