## Arrays [**Back**](./../README.md)

#### 1. Use `[]`

- To use literal syntax for array creation.
- Eslint rules tags: [`no-array-constructor`](http://eslint.org/docs/rules/no-array-constructor.html)

```js
/**
 * bad
 */
const items = new Array();

/**
 * good
 */
const items = [];
```
