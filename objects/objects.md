#### Use `{}`

- To use the literal syntax for object creation.
- Eslint rules tags: [`no-new-object`](http://eslint.org/docs/rules/no-new-object.html)

```js
/**
 * bad
 */
const item = new Object();

/**
 * good
 */
const item = {};
```
